## How to Upgrade

Follow these steps to update your Star Medical Wellness instance safely.

<Steps>
  <Step title="Backup Your Data" icon="database">
    Export all patient labs, hormone data, and treatment plans.

````bash
# Example backup command
starwellness backup --output=/path/to/backup.zip
````
  </Step>
  <Step title="Update Package" icon="download">
    Install the latest version via your package manager.

````bash
npm install star-medical-wellness@latest
# or
pip install star-medical-wellness==1.3.0
````
  </Step>
  <Step title="Run Migrations" icon="settings">
    Apply database schema changes.

````bash
starwellness migrate
````

    Review the changelog for breaking changes before proceeding.
  </Step>
  <Step title="Test Integrations" icon="check-circle">
    Verify API endpoints and third-party lab syncs work as expected.
  </Step>
</Steps>
